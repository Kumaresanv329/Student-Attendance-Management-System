# Bharathiar University Logo Setup

## ✅ Current Status:
- ✅ SVG placeholder logo created (`logo.svg`) - **Already visible on website!**
- ✅ Code configured to automatically display logo
- ⏳ Ready to replace with official PNG logo when available

## 📋 Quick Steps to Add Official Logo:

1. **Get the Official Logo File:**
   - Download from Bharathiar University official website
   - Or use the official logo image file you have

2. **Save the Logo:**
   - Save the file as `logo.png` (or `logo.jpg` if it's a JPG)
   - Place it in this folder: `frontend/public/logo.png`

3. **That's it!** The logo will automatically appear on:
   - Login page (centered at the top, with beautiful styling)
   - Main dashboard header (left side, professional appearance)

## File Formats Supported:
- ✅ PNG (recommended - best quality)
- ✅ JPG/JPEG
- ✅ SVG (scalable vector graphics)

## Recommended Specifications:
- **Format:** PNG with transparent background
- **Size:** 200-300px height (width will auto-adjust)
- **Resolution:** High resolution for best quality

## If Logo File is Missing:
- A fallback "BU" badge will be shown
- No errors will occur
- You can add the logo anytime

## 🎨 Logo Features:
The logo includes:
- Orange gear-shaped emblem with 16 teeth
- Central figure of Subramania Bharathiar
- Golden fish (left) and golden tiger (right)
- Stack of books (red, green, blue)
- Golden lamp (diya) below books
- Tamil text: "பாரதியார் பல்கலைக்கழகம், கோயம்புத்தூர்"
- English text: "BHARATHIAR UNIVERSITY"
- Blue banner with motto: "EDUCATE TO ELEVATE"

## 📍 Where to Get the Official Logo:
1. **Bharathiar University official website** (www.b-u.ac.in)
2. **University administration office**
3. **Your existing logo image file**
4. **University branding/marketing department**

## 💡 How It Works:
- System tries `logo.png` first (official logo)
- Falls back to `logo.svg` if PNG not found (current placeholder)
- Shows "BU" badge if neither exists (safety fallback)

**The placeholder SVG logo is already working and visible on your website!**

Once you place the official `logo.png` file in this folder, it will automatically replace the placeholder.
